## Cómo cambiar ApiKey ¿Cómo?
Cómo cambiarlo es bastante fácil, solo necesita cambiar la sintaxis a continuación
```js
if(apikeyInput != 'Nombre_de_la_APIKEY') return res.json(loghandler.invalidKey)
```
